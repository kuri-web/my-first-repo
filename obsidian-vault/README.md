# My Knowledge Vault - フォルダ構造

## 📁 フォルダ構成

```
MyVault/
├── 00_Inbox/          # 未整理の一時保存（毎週レビュー）
├── 10_Web/            # Webサイト・記事・ブログのメモ
├── 20_X_Twitter/      # X (Twitter) のツイート・スレッドメモ
├── 30_YouTube/        # YouTube 動画のメモ
├── 40_Ideas/          # 自分のアイデア・思考メモ
├── 50_Projects/       # プロジェクト・タスク管理
├── 90_Archive/        # 古いノート・不要になったもの
└── _templates/        # ノートテンプレート（このフォルダ）
    ├── web-article.md
    ├── x-twitter.md
    ├── youtube.md
    └── idea.md
```

## 🏷 タグ設計

### タイプタグ（必須）
- `#web` `#twitter` `#youtube` `#idea`

### トピックタグ（任意・自由に追加）
- `#テクノロジー` `#ビジネス` `#マーケティング` `#学習` `#健康` など

### ステータスタグ（アイデアノートのみ）
- `#草案` `#検討中` `#実行中` `#完了`
